### **Usage Examples:**

```bash
# 1. Analyze a single file with all approaches
python debug_ass.py analyze mycode.php --format html --output report.html

# 2. Debug pattern matching
python debug_ass.py debug patterns/php_syntax.json myscript.php

# 3. Batch analyze a project
python debug_ass.py batch ./project/ --output ./reports/ --format json

# 4. Test specific approach
python debug_ass.py approach cognitive --file complex.js
python debug_ass.py approach wavelet --file signal_processing.py
python debug_ass.py approach genetic --file optimized.js

```
